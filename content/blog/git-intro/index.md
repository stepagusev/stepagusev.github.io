В этом посте я расскажу, что такое git и зачем он нужен. Git - это система контроля версий, которая позволяет отслеживать изменения в коде и работать над проектами совместно.

## Основные команды git

- 'git init'   - создать новый репозиторий
- 'git clone'  - скопировать удалённый репозиторий
- 'git add'    - добавить изменений в индекс
- 'git commit' - зафиксировать изменения
- 'git push'   - отправить изменения на сервер
- 'git pull'   - скачать обновления с сервера

## Пример рабочего процесса

1. Вносим изменения в файлы
2. 'git add .'
3. 'git commit - m "Описание изменений"'
4. 'git push'
